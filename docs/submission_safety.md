# Submission Safety

Hackathon War Room uses synthetic demo data only.

## Do Not Include

- API keys
- Passwords
- Tokens
- Customer data
- Personally identifiable information
- Confidential company data
- Private repository contents
- Production logs
- Pre release product information under NDA

## Safety Posture

The project is designed to run locally with deterministic sample data for reliable judging. Sensitive configuration should be stored in environment variables and excluded from Git.

## Before Submission

Run the smoke test, review Git status, inspect commit history, and scan for common secret markers before publishing.
